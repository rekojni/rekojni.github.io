Learning & Development Tracking Dashboard

Overview

Created centralised learning dashboard to track training progress across department. Consolidated fragmented data sources and automated monitoring for 100+ staff members, improving compliance visibility and decision-making.

The Challenge

Department lacked unified learning progress tracking system:

  • Fragmented data: Learning status scattered across multiple systems
  • Manual tracking: Error-prone spreadsheet updates
  • Limited oversight: Difficult to identify training gaps or monitor compliance
  • Reactive planning: No timely insights for strategic learning alignment

The Solution

Developed automated dashboard using Excel, Power Query, SharePoint, and Microsoft Lists. Integrated critical data sources into single point of reference.

  • Automated Data Refresh: Power Query was used to fetch and clean data, replacing manual inputs and ensuring more timely updates.

    • Key Features:

    • Automated data integration: Power Query pulls from multiple sources
    • Role-based categorisation: Mandatory, role-specific, and elective learning tracks
    • Visual progress tracking: Conditional formatting and completion metrics
    • Justification capture: Microsoft List linked to dashboard for accountability
